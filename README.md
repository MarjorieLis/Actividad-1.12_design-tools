# DesignTools - Sistema de Temas Adaptativos

Proyecto de Interfaces y Multimedia: Implementación de sistema visual con cambio dinámico entre tema claro y oscuro.

## Descripción

DesignTools es una aplicación ficticia de diseño básico que demuestra la implementación de un sistema de temas completo. El usuario puede alternar entre modo claro y oscuro según sus preferencias.

## Características

- Cambio instantáneo entre tema claro y oscuro
- Persistencia del tema usando localStorage
- Transiciones suaves en todos los elementos
- Componentes UI completos:
  - Barra de navegación
  - Tarjetas interactivas
  - Botones en diferentes estados (normal, hover, disabled)
  - Formularios con inputs, select y slider
- Accesibilidad: contraste WCAG 2.1 nivel AA

## Uso

1. Abre `index.html` en cualquier navegador moderno
2. Usa el switch para alternar entre modo claro y oscuro
3. El tema seleccionado se guarda automáticamente

## Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Variables, Grid, Flexbox, Transiciones
- **JavaScript** - Vanilla JS (sin frameworks)
