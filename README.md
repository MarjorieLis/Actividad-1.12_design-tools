# DesignTools - Sistema de Temas

Proyecto de Interfaces y Multimedia: Sistema visual con cambio entre tema claro y oscuro.

## Descripción

DesignTools es una aplicación ficticia de diseño básico que demuestra la implementación de un sistema de temas.

## Características

- Cambio entre tema claro y oscuro
- Guarda la preferencia del usuario
- Componentes UI básicos:
  * Barra de navegación
  * Tarjetas de herramientas
  * Botones en diferentes estados
  * Formulario de configuración

## Paleta de colores

### Tema Claro
- Primario: #4A6CF7 (azul)
- Secundario: #2D9CDB (azul claro)
- Acento: #F46A6A (rojo)
- Fondo: #F5F5F5
- Texto: #222222

### Tema Oscuro
- Primario: #5A77F5 (azul claro)
- Secundario: #3DA5D9 (azul más claro)
- Acento: #F46A6A (rojo)
- Fondo: #222222
- Texto: #E0E0E0


## Notas

- El diseño es intencionalmente simple
- Algunos elementos no están perfectamente alineados
- Los colores son básicos y no están perfectamente calculados
- La implementación es funcional pero no es la más eficiente
